# Floaters

Floaters is a background-running radial launcher application for Windows.

## Features
- Summon the launcher from anywhere on your screen with a single hotkey.
- Quickly launch applications using intuitive mouse operations.
- Easily customizable, allowing you to change displayed apps via text files and configure animations via an INI file.

## How to Use
1. Run `Floaters.exe` (it stays active in the background).
2. **Double-click Shift and Ctrl simultaneously** to display the launcher.
3. **Double-click Shift and Ctrl simultaneously** again to hide the launcher.
4. To completely close the application, exit `Floaters.exe` from the system tray (indicator).

## Configuration

### Registering Applications 
The applications displayed in the launcher can be edited from the settings by clicking the system tray indicator.

## How to Build
This application is written in C++ and compiled using the MSVC (Microsoft Visual C++) compiler.

Please ensure that Visual Studio, including C++ build tools, is installed on your system.
